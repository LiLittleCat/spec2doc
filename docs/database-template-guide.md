# 数据库模板指南

本文档说明数据库文档模板的当前能力与建议用法。

## 1. 当前能力说明

当前版本中，数据库文档流程主要用于结构解析与界面预览，模板部分仍以内置样式为主。  
如果你选择自定义数据库模板，建议先复制内置模板进行样式调整（字体、颜色、页眉页脚、表格样式）。

## 2. 推荐修改范围

- 封面与标题样式
- 各级标题样式（Heading 1/2/3）
- 表格样式（边框、表头底色、列宽）
- 页眉页脚内容
- 段落间距和缩进

## 3. 模板编辑建议

1. 使用 `.docx` 格式保存
2. 保留原有章节结构，先改样式再改内容
3. 修改后先用一份小型 DDL/数据库结构做快速验证
4. 若出现排版错位，优先检查表格自动换行与列宽策略

## 4. 最佳实践

- 表格列宽固定，避免长字段名挤压布局
- 表头与正文保持清晰对比
- 大段说明采用统一段落样式，避免混合格式
- 模板版本建议按日期管理，例如：`数据库模板-20260206.docx`
